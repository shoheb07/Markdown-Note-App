# Markdown-Note-App
Markdown Note App is a web application that allows users to write notes in Markdown format and see a live preview of the rendered content instantly. It is useful for developers, students, bloggers, and technical writers.


## Overview

Markdown Note App is a web application that enables users to write notes using Markdown syntax and view the rendered output in real time.

The project provides a split-screen interface where users can edit notes on one side and preview formatted content on the other.

## Features

- Live Markdown Preview
- Auto Save Notes
- Responsive Layout
- Local Storage Support
- Beginner-Friendly Design

## Technologies Used

- HTML
- CSS
- JavaScript
- Marked.js

## Installation

No installation required.

Simply open:

index.html

in your browser.
